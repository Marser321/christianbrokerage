# Christian Brokerage — Web App

Sitio web de **Christian Brokerage USA**: asesoría en español para seguros, taxes e inmigración en Nueva York.

## Stack

- React 19 + TypeScript
- Vite 8
- Tailwind CSS 4
- Framer Motion + GSAP
- Desplegado en **Vercel**

## Desarrollo local

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build   # genera /dist
```

El deploy se maneja automáticamente vía Vercel al pushear a `main`.
